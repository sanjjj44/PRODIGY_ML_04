✋ Hand Gesture Recognition System using Deep Learning

📌 Project Overview

This project aims to develop a Hand Gesture Recognition System capable of identifying and classifying different hand gestures from image data using Deep Learning techniques.

The system enables intuitive human-computer interaction, where gestures can be interpreted as commands or inputs.

A custom dataset was used for training the model to simulate real-world implementation scenarios.




🎯 Objective

To build a deep learning model that accurately recognizes hand gestures from images and classifies them into predefined gesture categories.




📂 Dataset

✅ Custom Dataset Used

Instead of using the provided dataset, hand gesture images were independently collected and organized manually.

Dataset Details:

Self-collected gesture images

Multiple gesture classes

Folder-wise class labeling

Images resized and normalized before training





⚙️ Technologies Used

Python

TensorFlow

Keras

OpenCV

NumPy

Matplotlib





🧠 Deep Learning Approach

Convolutional Neural Network (CNN)

Image Classification

Supervised Learning


CNN helps the model automatically learn visual patterns such as edges, shapes, and gesture structures from images.




🔄 Workflow

1. Imported required libraries


2. Loaded custom gesture dataset


3. Image preprocessing and normalization


4. Dataset splitting (training & validation)


5. Designed CNN architecture


6. Model training using multiple epochs


7. Accuracy and loss evaluation


8. Gesture prediction






📈 Model Output

The trained model successfully classifies hand gestures based on learned visual features.
Training accuracy and validation performance were analyzed to evaluate model effectiveness.
